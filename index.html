批次文字轉 QR Code 工具

將文字內容自動生成 QR Code

使用說明：

  • 在下方文字區域貼上您要轉換的文字內容
  • 系統會自動偵測每一行或「標籤: 內容」格式的項目
  • 每個偵測到的項目會自動編號並生成對應的 QR Code
  • 可以為內容加上標籤，格式：標籤名稱: 您的內容
  • 支援批次下載所有 QR Code 為 ZIP 檔案
支援格式：
• 標籤名稱: 您的內容
• 直接貼上任意文字行
• 混合文字中的有效內容也會被偵測
**注意：程式現在會將「 ** ** 」作為項目之間的分隔符號。
